# Erase-COMDB
執行方式：
- 使用解壓縮軟體將其解壓縮(7Zip、WINRAR)。
- 將"Admin清除ComDB"以右鍵管理員執行。
- 雙擊"Admin清除ComDB - 捷徑"清除Com port。
- 執行USBDEVIEW 6.4 點選Vendor ID或Product ID排序，按住左鍵框選反藍大量異常Enum框選後點右鍵Uninstall。
